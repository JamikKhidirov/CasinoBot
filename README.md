# SuperBot 🤖

Многофункциональный Telegram бот с **казино** (PVP, solo vs bot, Blackjack), **OSINT-пробивом**, **анонимным чатом**, **модерацией** и **админ-панелью**.

## 📋 Команды

### 🎰 Казино
| Команда | Описание | Где работает |
|---------|----------|-------------|
| `/игры` или `/games` | Список игр | Везде |
| `/профиль` или `/profile` | Профиль и балансы | ЛС |
| `/топ` или `/top` | Топ игроков PVP | Везде |
| `/активные` или `/active` | Активные игры | Везде |
| `/разблокировать` или `/unlock` | Отменить свои игры | Везде |
| `/solotop` | Топ соло-казино | Везде |

#### 👥 PVP (с игроками, только в группах)
| Команда | Описание |
|---------|----------|
| `/куб [ставка]` | Кости 🎲 |
| `/боулинг [ставка]` | Боулинг 🎳 |
| `/дротики [ставка]` | Дротики 🎯 |
| `/баскетбол [ставка]` | Баскетбол 🏀 |
| `/футбол [ставка]` | Футбол ⚽ |

#### 🤖 С ботом (только в ЛС)
| Команда | Описание |
|---------|----------|
| `/сботом [игра] [ставка]` | Игра против бота, своя валюта `bot_balance` |

Пример: `/сботом куб 50`

#### 🃏 Блэкджек (только в группах)
| Команда | Описание |
|---------|----------|
| `/блекджек [ставка]` | Создать стол (до 6 игроков, каждый против дилера) |

### 🎮 Меню казино
В `/игры` доступны 3 режима:
- **🤖 Игра с ботом** — в ЛС, бот кидает кубик одновременно, своя валюта
- **👥 С игроками** — в группе, PVP, создание комнаты, таймер 30 сек
- **🃏 Блэкджек** — в группе, до 6 игроков, кнопки Hit/Stand

### 🛡 Модерация

## 🚀 Установка и запуск

```bash
# Клонирование
git clone <repo-url> superbot
cd superbot

# Зависимости
pip install -r requirements.txt

# Настройка
# Отредактировать config.py -> BOT_TOKEN, OWNER_ID

# Запуск
python main.py
```

## ⚙️ Конфигурация

`config.py`:
- `BOT_TOKEN` — токен бота (Telegram @BotFather)
- `OWNER_ID` — ваш Telegram ID (разработчик)
- `ADMINS` — список ID администраторов
- `VERSION` — версия бота
- `PROXY_URL` — прокси (если Telegram заблокирован)

## 🔐 Права доступа

| Право | Описание |
|-------|----------|
| `view_players` | Список игроков |
| `view_stats` | Статистика |
| `add_balance` | Пополнение баланса пользователям |
| `approve_deposits` | Одобрение пополнений |
| `approve_withdrawals` | Одобрение выводов |
| `create_promos` | Создание промокодов |
| `manage_admins` | Управление админами |
| `manage_games` | Управление играми |

## 🎲 Система балансов

У каждого игрока 3 независимых баланса:
- **💰 PVP баланс** (`balance`) — для игр с другими игроками
- **🤖 Баланс с ботом** (`bot_balance`) — для игр против бота в ЛС
- **🃏 Баланс блэкджека** (`blackjack_balance`) — для блэкджека

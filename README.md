# SuperBot 🤖

Многофункциональный Telegram бот с **казино** (PVP, solo vs bot, Blackjack), **OSINT-пробивом**, **анонимным чатом**, **модерацией** и **админ-панелью**.

## 📋 Команды

### 🎰 Казино

#### 👥 PVP (с игроками, только в группах)
| Команда | Описание | Пример |
|---------|----------|--------|
| `/куб [ставка]` | Кости 🎲 — кто выбросит больше | `/куб 100` |
| `/боулинг [ставка]` | Боулинг 🎳 — сбиваешь кегли | `/боулинг 50` |
| `/дротики [ставка]` | Дротики 🎯 — кто ближе к яблочку | `/дротики 200` |
| `/баскетбол [ставка]` | Баскетбол 🏀 — >3 = попадание | `/баскетбол 150` |
| `/футбол [ставка]` | Футбол ⚽ — >3 = гол | `/футбол 100` |

Без ставки — запросит выбор, либо можно кинуть эмодзи 🎲🏀⚽🎳🎯 в ответ на игру.

> **Правила PVP:** Первый игрок создаёт игру, второй присоединяется (кнопка), таймер 30 сек на бросок. Если второй не кинул — первый забирает ставку. Победа = +ставка.

#### 🤖 С ботом (только в ЛС)
| Команда | Описание |
|---------|----------|
| `/сботом [игра] [ставка]` | Игра против бота, своя валюта `bot_balance` |

Пример: `/сботом куб 50` — бот автоматически кидает свой кубик.

#### 🃏 Блэкджек (только в группах)
| Команда | Описание |
|---------|----------|
| `/блекджек [ставка]` | Создать стол (до 6 игроков, каждый против дилера) |

> **Правила:** Каждому раздаётся по 2 карты. Дилер набирает до 17+. Если у игрока перебор (>21) — проигрыш, если у дилера перебор — выигрыш, иначе сравниваются суммы. Blackjack (21 с первых 2 карт) = победа ×1.5.

#### Общие команды казино
| Команда | Описание | Где работает |
|---------|----------|-------------|
| `/профиль` или `/profile` | Профиль, балансы, аватар | Везде |
| `/игры` или `/games` | Меню выбора режима | Везде |
| `/топ` или `/top` | Топ-10 по PVP балансу | Везде |
| `/активные` или `/active` | Список текущих игр | Везде |
| `/разблокировать` или `/unlock` | Принудительно отменить свои игры | Везде |
| `/solotop` | Топ-10 по игре с ботом | Везде |
| `/бонус` | Ежедневный бонус (100 монет PVP) | Везде |

### 🎮 Меню казино (`/игры`)
3 режима выбора:
1. **🤖 Игра с ботом** — `/сботом` в ЛС
2. **👥 С игроками** — PVP команды в группах
3. **🃏 Блэкджек** — `/блекджек` в группах

Если PVP или блэкджек вызваны в ЛС — бот пришлёт инструкцию как добавить его в группу.

### 🔍 OSINT (только для админов)
| Команда | Описание | Источник |
|---------|----------|----------|
| `/phone +71234567890` | Инфо о номере (страна, оператор, регион) | phonenumbers lib |
| `/hackphone +71234567890` | Расширенный скан номера | phonenumbers lib |
| `/email user@example.com` | Валидация email + MX + Gravatar | DNS/HTTP |
| `/user username` | Поиск username в 22 соцсетях | HTTP (sherlock-like) |
| `/ip 1.2.3.4` | Геолокация IP | ip-api.com |
| `/domain example.com` | DNS A/AAAA/MX/NS/TXT/SOA + HTTP/HTTPS | DNS/HTTP |
| `/card 123456******7890` | Проверка BIN карты | bins. su |
| `/wifi SSID` | Анализ Wi-Fi сети | wigle.net |

Также доступно через меню `/start` → **OSINT-пробив**.

### 🛡 Модерация анонимного чата
| Команда | Описание | Формат |
|---------|----------|--------|
| `/mod` | Панель модерации с кнопками | — |
| `/ban @user 123` или `/ban 123456` | Забанить | `/ban id/username [причина]` |
| `/unban` | Разбанить | `/unban id/username` |
| `/mute` | Замутить на N минут | `/mute id/username минуты` |
| `/unmute` | Размутить | `/unmute id/username` |
| `/warn` | Выдать варн (3/3 = автомат бан) | `/warn id/username [причина]` |
| `/warns` | Проверить варны | `/warns id/username` |
| `/check` | Полная информация о пользователе | `/check id/username` |
| `/chatlog` | История переписки (лог) | `/chatlog id` или `/chatlog id1 id2` |

Все команды работают как по **числовому ID**, так и по **@username** (разрешается через `users` таблицу).

## 💾 Базы данных

### `chat.db` (основная SQLite, инициализируется в `db.py`)
| Таблица | Назначение | Ключевые колонки |
|---------|-----------|------------------|
| `users` | Все пользователи | `user_id`, `username`, `balance`, `bot_balance`, `blackjack_balance`, `reg_date` |
| `bans` | Баны | `user_id`, `admin_id`, `reason`, `date` |
| `messages` | История сообщений чата | `sender_id`, `receiver_id`, `message`, `timestamp` |
| `osint_logs` | Логи OSINT запросов | `user_id`, `type`, `query`, `timestamp` |
| `reports` | Жалобы | `reporter_id`, `target_id`, `reason`, `timestamp` |
| `casino_admins` | Админы казино | `user_id`, `can_*` (8 прав) |
| `promocodes` | Промокоды | `code`, `amount`, `active` |
| `transactions` | Пополнения/выводы | `user_id`, `amount`, `type`, `status`, `admin_id` |
| `casino_deposits` | Запросы пополнения | `user_id`, `amount`, `status` |
| `casino_withdrawals` | Запросы вывода | `user_id`, `amount`, `status` |

### `casino.db` (standalone CasinoBot)
Отдельный SQLite для старого standalone бота (`casino/casino.py`).

## 🚀 Установка и запуск

```bash
# 1. Клонирование
git clone <repo-url> superbot
cd superbot

# 2. Зависимости
pip install -r requirements.txt

# 3. Настройка config.py
#    - BOT_TOKEN = "..."
#    - OWNER_ID = ваш Telegram ID
#    - ADMINS = [OWNER_ID]
#    - PROXY_URL = "" (если нужно)

# 4. Запуск
python main.py
```

### Хостинг на amvera
Проект содержит `amvera.yml` с настройками деплоя:
```yaml
meta:
  environment: python
  scriptName: main.py
```
Просто подключите репозиторий в amvera → бот запустится автоматически.

## 🎲 Система балансов

У каждого игрока **3 независимых баланса**:

| Баланс | Поле в БД | Начальное значение | Для чего |
|--------|-----------|-------------------|----------|
| 💰 PVP | `balance` | 1000 | Игры с другими игроками в группах |
| 🤖 С ботом | `bot_balance` | 500 | `/сботом` в ЛС |
| 🃏 Блэкджек | `blackjack_balance` | 1000 | `/блекджек` в группах |

Обнуление/пополнение каждого баланса — через админ-панель.

## 🎲 Детали игр

### Кости 🎲
Бот кидает два кубика (1-6). У кого сумма больше — тот победил. Ничья — возврат ставки.

### Боулинг 🎳
Сбиваешь виртуальные кегли. Счёт игрока vs счёт оппонента.

### Дротики 🎯
Бросок дротика: кто ближе к центру (по случайному числу).

### Баскетбол 🏀
Кидаешь мяч: если выпало **>3** — попадание, ≤3 — промах.

### Футбол ⚽
Бьёшь по воротам: если выпало **>3** — гол, ≤3 — промах.

### Блэкджек 🃏
Колода 52 карты, значения: 2-10 по номиналу, J/Q/K = 10, A = 1 или 11.
- Игроку и дилеру раздаётся по 2 карты
- Можно Hit (взять) или Stand (остановиться)
- Перебор (>21) — проигрыш
- Дилер добирает до 17+
- Blackjack (21 с первых двух карт) — победа ×1.5

## 💬 Анонимный чат
- `/start` → «Поиск собеседника» — ставит в очередь
- Когда находится пара — начинается анонимная переписка
- Каждое сообщение пересылается через бота (получатель видит только текст)
- Жалобы и модерация через администраторов

## 📝 Логирование
Ошибки пишутся в `bot_errors.log`. При превышении ответа 4096 символов обрезается до 3997 + "...". Все HTTP-запросы OSINT имеют таймаут 8-15 секунд.

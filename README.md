# SuperBot 🤖

Многофункциональный Telegram бот с казино, OSINT-пробивом, модерацией и админ-панелью.

## 📋 Команды

### 🎰 Казино
| Команда | Описание |
|---------|----------|
| `/profile` | Профиль и баланс |
| `/top` | Топ игроков |
| `/dice [ставка]` | Кости |
| `/bowling [ставка]` | Боулинг |
| `/darts [ставка]` | Дротики |
| `/basket [ставка]` | Баскетбол |
| `/football [ставка]` | Футбол |
| `/games` | Список игр |
| `/active` | Активные игры |
| `/unlock` | Отменить свои игры |
| `/promo КОД` | Активировать промокод |

### 🔍 OSINT (админы)
| Команда | Описание |
|---------|----------|
| `/phone +71234567890` | Пробив номера |
| `/hackphone +71234567890` | Скан номера |
| `/email email@example.com` | Пробив email |
| `/user username` | Поиск username |
| `/ip 1.2.3.4` | Геолокация IP |
| `/domain example.com` | Инфо домена |
| `/card 123456xxxxxx` | Пробив карты |
| `/wifi BSSID/SSID/IP` | Анализ Wi-Fi |

### 👑 Админ-панель
| Команда | Описание |
|---------|----------|
| `/admin` | Панель казино |
| `/players` | Список игроков |
| `/stats` | Статистика |
| `/setperm user_id право` | Выдать право |
| `/addadmin id` | Добавить админа |
| `/createpromo код сумма` | Создать промокод |
| `/deletepromo код` | Удалить промокод |
| `/promo_list` | Список промокодов |

### 🛡 Модерация
| Команда | Описание |
|---------|----------|
| `/ban` | Забанить |
| `/unban` | Разбанить |
| `/mute` | Замутить |
| `/unmute` | Размутить |
| `/warn` | Варн |
| `/check` | Проверить |
| `/chatlog` | Переписка |

## 🚀 Установка и запуск

```bash
# Клонирование
git clone <repo-url> superbot
cd superbot

# Зависимости
pip install -r requirements.txt

# Настройка
# Отредактировать config.py -> BOT_TOKEN

# Запуск
python main.py
```

## ⚙️ Конфигурация

`config.py`:
- `BOT_TOKEN` — токен бота (Telegram @BotFather)
- `OWNER_ID` — ваш Telegram ID (разработчик)
- `API ключи` — опционально (можно оставить пустыми)

## 🔐 Права доступа

| Право | Описание |
|-------|----------|
